## Selected Projects

### [Semantic Risk Modelling for Autonomous Navigation](https://github.com/daquinga2020/TFM-2026)

Collaborative MSc thesis project developed with ROS 2, Gazebo, Nav2 and TurtleBot4.

My contribution focuses on the representation and publication of semantic dynamic entities for their subsequent integration into a risk-aware navigation system.

Main contributions:

* Custom ROS 2 messages for semantic entities
* C++ semantic entity provider
* Dynamic pose extraction from Gazebo
* Velocity estimation for moving entities
* Configurable entities for different simulated environments
* Position-noise experiments
* ROS bag recording and reproducible execution scripts

### [Machine Learning Case Studies](https://github.com/CGiral/machine-learning-case-studies)

Collection of supervised machine learning projects covering regression, classification, model evaluation, hyperparameter optimisation and imbalanced datasets.

### [Unsupervised Learning Case Studies](https://github.com/CGiral/unsupervised-learning-case-studies)
